# Iteraciones

Un archivo por funcionalidad construida con la skill `/modulo`.

Nomenclatura: `YYYY-MM-DD-<slug-modulo>.md`

Estados posibles:
- `en captura` — fase 1, recolectando requerimiento.
- `esperando revisión` — implementado, esperando feedback del usuario.
- `cerrada` — revisado y con mejora aplicada (si correspondía).

Estos archivos son la memoria persistente del avance del proyecto entre conversaciones.
